Instructions

Many times an application needs to verify the account by sending a verification code by text message.

Lets create a functionality the enables the user to paste the verification code to the input fields.

Features

  • The user is allowed to type the values manually. After each value is inserted, the next input will be focused.
  • . The user is allowed to paste the verification code.
  • An extra challenge will be to auto submit the form once all inputs all populated.

Things to thing about:

What happens if the user pastes only 3 digits and there are 6 inputs to fill out.

Example

Things to help you:

Google “paste event” to understand how to listen to a paste event

Google “clipboardData” to find out how to get the value of a paste event.
